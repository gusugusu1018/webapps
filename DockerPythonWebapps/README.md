# flask on docker (test environment for azure web apps)
## 1. install docker
read this link  
[Official site](https://docs.docker.com/engine/installation/)

## 2. docker build
```
make build
```
in image directory

## 3. docker run
```
make run
```
in image directory

## 3. docker stop && docker rm
```
make rm
```
in image directory
